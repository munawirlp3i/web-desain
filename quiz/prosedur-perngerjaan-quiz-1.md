## 🛠️ LANGKAH 1: PERSIAPAN & MEMBACA SOAL

1. Silakan buka browser di PC Lab kalian masing-masing.
2. Akses repositori GitHub Dosen melalui link berikut:
👉 **`(https://github.com/munawirlp3i/web-desain/tree/main/quiz)`**
3. Buka folder `quiz`, lalu lihat petunjuk tampilan soal pada file **`soal-quiz-1.pdf`** atau **`soal-quiz-1.png`**.
4. Amati kerangka halaman, tabel kegiatan kampus, dan formulir pendaftaran yang harus kalian kloning (buat ulang).

---

## 📂 LANGKAH 2: SETUP FOLDER & STRUKTUR FILE (WAJIB!)

Kalian ditantang untuk menulis kode secara bersih menggunakan metode **External CSS** (file terpisah). Ikuti struktur folder berikut:

1. Di dalam folder kerja utama kalian, buatlah folder baru bernama **`quiz`**.
2. Di dalam folder `quiz` tersebut, buat **2 file wajib**:
* **`index.html`** (Untuk kerangka HTML dan Form).
* **`style.css`** (Untuk menghias tampilan).


3. Hubungkan file `index.html` kalian ke `style.css` menggunakan tag `<link rel="stylesheet" href="style.css">` di dalam elemen `<head>`.
4. *Selamat ngoding! Selalu cek tampilan web kalian secara berkala menggunakan ekstensi **Live Server**.*

---

## 🚀 LANGKAH 3: CARA PENGUMPULAN & UPLOAD HASIL (PILIH SALAH SATU)

Setelah coding selesai dan web sudah rapi, kalian **wajib mengumpulkan** hasil pengerjaan ke akun GitHub masing-masing. Silakan pilih salah satu dari 2 metode di bawah ini:

### 🔹 CARA 1: Menggunakan Git GUI di VS Code (Sangat Direkomendasikan/Gaya Pro)

*Pastikan kalian sudah menyiapkan Token Akses (PAT) GitHub masing-masing.*

1. **Staging Files:** Source Control Tab.
Buka tab **Source Control** (ikon cabang ketiga dari atas di sebelah kiri VS Code). Klik tanda tambah (**`+`**) atau *Stage All Changes* di samping file `index.html` dan `style.css`.


2. **Commit:** Beri Catatan.
Ketik pesan commit pada kolom yang tersedia, contoh: **`"Quiz 1: Selesai Pengerjaan"`**, lalu klik tombol centang (**Commit**).


3. **Push ke GitHub:** Selesai.
Klik tombol **Sync Changes** atau klik ikon titik tiga (`...`) lalu pilih **Push**. Jika diminta password, masukkan **Username** dan **Token PAT** kalian. Selesai!


### 🔹 CARA 2: Upload Manual via Website GitHub (Opsi Cadangan/Alternatif)

*Gunakan cara ini jika Git lokal di PC Kampus kalian mengalami kendala atau error koneksi.*

1. **Buka GitHub:** Lewat Browser.
Buka browser, login ke akun GitHub kalian, dan masuk ke repositori tugas web dasar kalian masing-masing.


2. **Pilih Fitur Upload:** Add File.
Di halaman utama repositori kalian, klik tombol **Add file** di pojok kanan atas, lalu pilih **Upload files**.


3. **Drag & Drop Folder:** Proses Upload.
Tarik (*Drag & Drop*) folder **`quiz`** (yang berisi `index.html` dan `style.css`) dari File Explorer komputer kalian ke dalam kotak upload di browser. Tunggu hingga proses upload selesai.


4. **Commit Changes:** Simpan.
Di bagian bawah halaman, ketik judul commit (misal: `"Upload Quiz 1"`) lalu klik tombol hijau **Commit changes**. Selesai!

## 🛑 SOP PENTING SEBELUM MENINGGALKAN LAB!

Karena PC ini digunakan bersama mahasiswa lain, demi **KEAMANAN AKUN** kalian, wajib lakukan 3 hal ini sebelum pulang:

1. **VS Code:** Klik ikon akun di pojok kiri bawah -> Pilih **Sign Out** dari GitHub.
2. **Windows Credential:** Klik *Start*, cari "Credential Manager" -> Pilih *Windows Credentials* -> Cari `git:(https://github.com)` -> Klik **Remove**.
3. **Browser:** Pastikan sudah **Log Out** dari situs `github.com`.
